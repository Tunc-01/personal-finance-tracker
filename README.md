# Personal Finance Tracker

A Python command-line personal finance tracker that stores transactions in a CSV file and generates reports.

## Features
- Add income/expense transactions
- Save data to `transactions.csv`
- View recent transactions
- Overall summary (income, expense, balance)
- Monthly report + drill-down details
- Date range report
- Demo data generator

## Files
- `finance_tracker.py` — main application
- `transactions.csv` — auto-created data file

## Requirements
- Python 3.x

## Run
```bash
python finance_tracker.py
